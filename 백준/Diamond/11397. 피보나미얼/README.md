### 입력 

 <p>첫 번째 줄에 두 자연수 n과 p (1 ≤ n ≤ 10<sup>9</sup>, 2 ≤ p ≤ 10<sup>3</sup>)이 공백을 사이로 두고 주어진다.</p>

### 출력 

 <p>p - 1줄에 걸쳐 답을 출력한다. 이 중 i(1 ≤ i ≤ p - 1)번째 줄에는 F<sub>n</sub>이 (i + 1)로 나누어 떨어지지 않도록 하기 위해 F<sub>n</sub>을 (i + 1)로 나눠야 할 횟수를 출력해야 한다.</p>
